# CHARUSAT Student ID Lookup

A fast, modern web tool to look up student names and IDs at CHARUSAT (Charotar University of Science & Technology).

**Can't remember a friend's ID? Or want to find who someone is from their ID? This is a tool for that.**

## Features

- 🔤 **Search by Name** — Fuzzy search with word-order-independent matching. Type any part of a name in any order.
- 🔢 **Search by ID** — Instant exact ID lookup.
- ⚡ **Instant results** — All data loads client-side, zero backend latency.
- 🎨 **Premium dark UI** — Clean, modern design with smooth animations.
- 📱 **Responsive** — Works on desktop and mobile.

## Tech Stack

- **Next.js** (App Router, static export)
- **Fuse.js** for fuzzy search
- **Vanilla CSS** with custom design system
- Deployed on **Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub and import in [Vercel](https://vercel.com) — it works out of the box with zero configuration.
